val fortify : Player.player -> int -> Territories.territory -> Territories.territory -> Territories.territory list -> Player.player

fortify p troop terr1 terr2 moves a given number of troops (troop) from one territory terr1 to another terr2 in the fortify phase. troop must be greater than 1. Raises CantUseToFortify if terr1 cannot provide troops. Raises InvalidTerritory if a territory cannot be fortified
